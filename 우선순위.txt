

# 1단계 - 필수 기반 기능 (P0)
1. 사용자 인증
- 회원가입 (이메일, 사용자명, 비밀번호)
- 로그인/로그아웃
- 비로그인 사용자 접근 제한
2. 기본 상품 관리
- 상품 등록 (이미지, 제목, 설명, 가격, 상태, 사이즈)
- 상품 목록 조회
- 상품 상세 보기
- 자신의 상품 수정/삭제

# 2단계 - 핵심 기능 (P1)
1.프로필 기능
- 기본 프로필 페이지 (사용자명, 가입일)
- 사용자가 등록한 상품 목록 표시
- 프로필 수정
2. 검색 기능
- 키워드 기반 상품 검색
- 기본 정렬 기능

# 3단계 - 소셜 기능 (P2)
1. 좋아요 기능
- 상품 좋아요/취소
- 좋아요한 상품 목록 표시
2. 팔로우 기능
- 사용자 팔로우/언팔로우
- 팔로워/팔로잉 수 표시
- 팔로우 목록 보기

# 제외 기능 (현재 MVP에서 제외)
- 구매/결제 시스템
- 위치 기반 필터링
- 알림 시스템
- 메시지 기능
- 리뷰/평점 시스템

# 구현 순서 제안:
1. Django 프로젝트 셋업 및 기본 인증
2. 상품 CRUD 기능
3. 프로필 페이지
4. 검색 기능
5. 좋아요 기능
6. 팔로우 기능

이러한 우선순위는 사용자가 기본적인 거래 플랫폼으로서의 기능을 먼저 사용할 수 있도록 하면서, 점진적으로 소셜 기능을 추가하는 방식으로 구성했습니다.
